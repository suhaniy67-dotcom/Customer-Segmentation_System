# 🛍️ Customer Segmentation Model for Marketing Campaigns

A Machine Learning-based Customer Segmentation System built using **K-Means Clustering** and **Streamlit**. This project helps businesses group customers into different segments based on purchasing behavior and demographics, enabling targeted marketing campaigns and better business decisions.

---

## 📌 Project Overview

Customer segmentation is one of the most important marketing strategies. This application uses **K-Means Clustering** to divide customers into meaningful groups based on their characteristics such as age, annual income, spending score, and purchase frequency.

The application provides an interactive dashboard where users can upload customer data, perform segmentation, visualize clusters, and download the results.

---

- 📂 Upload Customer Dataset (CSV)
